# پولاد سقف خلیج فارس

وب‌سایت معرفی محصولات، ثبت سفارش و مدیریت فروش تعاونی پولاد سقف خلیج فارس.

## فناوری‌ها

- Next.js 16 و React 19
- MongoDB و Mongoose
- NextAuth با ورود شماره تلفن و رمز
- shadcn/ui با Base UI
- Motion برای انیمیشن‌ها
- React Hook Form و Zod برای تمام فرم‌ها

## چرخه سفارش

1. کاربر محصول، مشخصات و وزن را انتخاب می‌کند.
2. محصول با قیمت معتبر سمت سرور به سبد خرید پیش‌نویس اضافه می‌شود.
3. کاربر می‌تواند وزن را تغییر دهد، محصول دیگری اضافه کند یا موردی را حذف کند.
4. با ثبت سبد، قیمت و موجودی دوباره بررسی و موجودی رزرو می‌شود.
5. مدیر یا کارمند، کل سبد را به‌صورت یک سفارش تایید، معلق یا لغو می‌کند.
6. در صورت لغو، موجودی رزروشده به محصول بازگردانده می‌شود.

## اجرا

فایل `.env.local.example` را به `.env.local` کپی کنید. در حالت توسعه و تست، برنامه از `MONGODB_URI_LOCAL` استفاده می‌کند و اگر این مقدار تنظیم نشده باشد به‌صورت پیش‌فرض به `mongodb://127.0.0.1:27017/poolad-saghf` متصل می‌شود. در محیط production مقدار `MONGODB_URI_PRODUCTION` استفاده می‌شود.

پس از تنظیم متغیرها:

```bash
pnpm install
pnpm dev
```

بررسی نهایی:

```bash
pnpm typecheck
pnpm lint
pnpm build
```
